## 가상 클래스 선택자
클래스 선택자처럼 행동하지만 클래스 선택자는 아닌 선택자
(엘리먼트의 특성에 따라 클래스 선택자처럼 몇몇 태그를 그룹핑한다.)
- hover, active, link, visited등(visited는 보안상의 문제 때문에 일부 속성만 사용 가능)
태그에서 위에 있는 것이 우선순위가 높다.
active가 제일 밑으로

:link - 방문한 적이 없는 링크
:visited - 방문한 적이 있는 링크
:hover - 마우스를 롤오버 했을 때 
:active - 마우스를 클릭했을 때
